# papers/

本地论文项目和参考资料。

## 目录约定

- 每个独立论文项目放在 `papers/<project-slug>/`
- 参考资料、课程数据源、外部样例放在 `papers/_reference/`
- 论文项目初始化：`/_template/` 复制到 `papers/<slug>/`，或使用 `/paper-workflow init`

## 注意事项

- `papers/` 下的内容默认属于本地工作区
- 提交前需确认是否包含：隐私数据、未发表论文、大文件、版权材料
- 论文的 `.paper-workflow/` 和 `outputs/` 会被 `.gitignore` 忽略
